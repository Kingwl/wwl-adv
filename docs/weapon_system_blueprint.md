# 武器系统蓝图

> 本文档是武器系统的策划设计蓝图，重点描述武器种类、战斗效果、流派标签和组合进化关系。
> 代码结构、数据字段和实现路径见 [`weapon_design.md`](./weapon_design.md)。
> 武器进入单局、升级池权重、宝箱和商店纠偏规则见 [`weapon_acquisition_design.md`](./weapon_acquisition_design.md)。

## 设计目标

- 长期完整蓝图：规划 `24` 把基础武器，支撑多局构筑和后续扩展。
- 混搭街机题材：允许冷兵器、魔法、道具、科技、召唤物共存，但要求单把武器视觉和玩法清晰。
- 清晰易懂：每把武器只有一个核心效果，玩家看到名称和描述就能理解用途。
- 保留当前基础：现有 `基础利刃`、`弓箭精通`、`天雷引`、`护盾球`、`荆棘护甲`、`生命源泉` 全部纳入蓝图。

## 核心规则

- 每局开局自带 1 把基础武器，后续通过升级解锁。
- 建议单局最多持有 `6` 把基础武器，避免升级池过度稀释。
- 每把基础武器最高 8 级，升级主要改变伤害、冷却、范围、数量、持续时间等可感知参数。
- 武器按体验分为 6 类：近身、弹幕、范围、控制、防御、辅助。
- 每把武器带 1-2 个流派标签，用于组织升级池、构筑方向和组合进化。
- 进化采用“组合进化”：主武器满级，且搭配武器已解锁并达到中等级后，升级池可出现进化选项。
- 进化会替换主武器，搭配武器保留；进化效果必须改变清怪方式，而不只是提高数值。

## 流派标签

| 标签 | 定位 | 典型体验 |
|------|------|----------|
| `斩击` | 近身清怪 | 玩家周围短距离高频伤害 |
| `弹体` | 远程输出 | 自动瞄准、穿透、散射、弹跳 |
| `范围` | 区域压制 | 对固定区域造成爆发或持续伤害 |
| `持续` | 长时间伤害 | 火场、毒雾、环绕物、陷阱 |
| `召唤` | 自动单位 | 无人机、炮塔等独立攻击源 |
| `控制` | 生存空间 | 减速、击退、吸附、冻结 |
| `防御` | 自保反击 | 护盾、反伤、近身保护 |
| `恢复` | 容错续航 | 治疗、吸血、溢出护盾 |
| `金币` | 经济转伤害 | 金币数量或消耗影响输出 |
| `爆发` | 瞬间清屏 | 高伤害、低频率、大视觉反馈 |

## 24 把基础武器

| 武器 | 分类/标签 | 核心效果 | 成长方向 |
|------|-----------|----------|----------|
| 基础利刃 | 近身 / 斩击 | 玩家周围圆形斩击，命中范围内所有敌人 | 伤害、范围、冷却 |
| 弓箭精通 | 弹幕 / 弹体 | 自动瞄准最近敌人发射箭矢 | 弹体数、射速、穿透 |
| 天雷引 | 范围 / 爆发 | 随机敌人位置落雷，造成小范围伤害 | 伤害、落雷次数、范围 |
| 护盾球 | 防御 / 持续 | 球体环绕玩家，碰撞敌人造成伤害 | 球数、半径、旋转速度 |
| 荆棘护甲 | 防御 / 反击 | 玩家受伤时对周围敌人反伤 | 反伤比例、范围、冷却保护 |
| 生命源泉 | 辅助 / 恢复 | 定时恢复生命 | 治疗量、冷却、溢出护盾 |
| 火焰瓶 | 范围 / 持续 | 向敌群投掷火瓶，地面留下燃烧区 | 燃烧持续、范围、伤害 |
| 冰霜环 | 控制 / 范围 | 周期性释放冰环，减速近身敌人 | 减速强度、范围、伤害 |
| 回旋镖 | 弹体 / 斩击 | 飞出后返回，往返都造成伤害 | 数量、飞行距离、宽度 |
| 散弹枪 | 弹幕 / 爆发 | 向最近敌人方向发射短程扇形弹幕 | 弹丸数、伤害、扩散角 |
| 毒液罐 | 范围 / 持续 | 命中后施加持续毒伤 | 毒伤、持续时间、扩散概率 |
| 地雷 | 控制 / 爆发 | 在玩家附近布置地雷，敌人靠近触发 | 地雷数、爆炸范围、伤害 |
| 激光笔 | 弹体 / 持续 | 周期性扫射一条直线激光 | 宽度、持续时间、转向速度 |
| 旋风陀螺 | 近身 / 持续 | 放出缓慢移动的旋风，持续推开敌人 | 数量、持续、击退 |
| 召唤无人机 | 召唤 / 弹体 | 小无人机自动追随并射击 | 无人机数、射速、伤害 |
| 玩具炮塔 | 召唤 / 防线 | 定点炮塔自动攻击附近敌人 | 炮塔数、持续、射程 |
| 黑洞弹 | 控制 / 范围 | 生成短暂吸附点，把敌人拉向中心 | 吸力、持续、范围 |
| 金币飞刃 | 金币 / 弹体 | 金币数量越高，周期性飞刃越强 | 飞刃数、金币倍率 |
| 幸运骰子 | 爆发 / 随机 | 随机触发小爆炸、治疗、金币或雷击 | 触发频率、效果强度 |
| 圣光棱镜 | 范围 / 恢复 | 对敌人造成伤害，同时少量治疗玩家 | 伤害、治疗、目标数 |
| 电磁链 | 弹体 / 控制 | 闪电在多个敌人之间弹跳 | 弹跳数、伤害、麻痹概率 |
| 锯片陷阱 | 近身 / 持续 | 锯片沿玩家周围短轨道来回切割 | 锯片数、轨道长度、伤害 |
| 音浪喇叭 | 控制 / 范围 | 周期性向外推波，击退并伤害敌人 | 击退、范围、冷却 |
| 火箭背包 | 移动 / 爆发 | 移动时周期性向身后喷火伤敌 | 喷射频率、火焰长度、伤害 |

## 流派方向

| 流派 | 推荐武器 | 体验目标 |
|------|----------|----------|
| 近身防御 | 基础利刃、护盾球、荆棘护甲、锯片陷阱 | 玩家可以贴近敌群走位，依靠环绕和反击打开空间 |
| 远程弹幕 | 弓箭精通、散弹枪、电磁链、激光笔 | 保持移动距离，用多弹体和穿透处理中远距离敌人 |
| 范围持续 | 火焰瓶、毒液罐、地雷、黑洞弹 | 在地图上制造危险区域，让敌人进入后持续掉血 |
| 召唤控制 | 召唤无人机、玩具炮塔、冰霜环、音浪喇叭 | 通过自动单位和控制效果减少玩家操作压力 |
| 恢复容错 | 生命源泉、圣光棱镜、荆棘护甲 | 用治疗和反击提高容错，但输出效率低于纯伤害流派 |
| 经济爆发 | 金币飞刃、幸运骰子、火箭背包 | 把金币或随机触发转化为高上限输出 |

## 成长原则

- 低等级强化要让玩家立刻感知：范围变大、弹体变多、冷却变短、视觉更明显。
- 高等级强化可以提高构筑方向：例如穿透、弹跳、持续时间、额外目标、溢出护盾。
- 防御和恢复武器提高生存能力，但不能替代主要输出武器。
- 金币和随机类武器允许收益更高，但要保留波动感和机会成本。
- 控制武器的目标是制造走位空间，不应无限定身或完全阻止敌人接近。
- 进化武器必须比基础武器更有辨识度，但不应该让其他武器完全失去意义。

## 验收标准

- 任意 3 把武器放在同一升级池中，玩家应能仅凭名称和描述理解效果差异。
- 至少能组成 4 种清晰流派：近身防御、远程弹幕、范围持续、召唤控制。
- 前 3 分钟基础武器能清小怪；中期靠组合形成方向；后期进化武器提供质变。
- 防御/恢复武器降低死亡风险，但不能替代主要输出。
- 每把武器需要一个可辨识视觉轮廓，避免多个武器都只是“圆形爆炸”。

## 设计假设

- 本文档只定义策划蓝图，不要求本轮实现全部武器。
- 命名以中文为主，允许混搭街机题材，不强制统一奇幻世界观。
- 当前 6 把武器全部保留，并作为未来完整系统的核心基础。
- 每把基础武器最多参与一个主要进化配方，降低记忆负担。
