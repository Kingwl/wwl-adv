# 游戏设计文档

> 游戏 UI 信息架构、HUD、升级选择、暂停和结算界面设计见 [`ui_design.md`](./ui_design.md)。
> 本地存档和继续游戏设计见 [`save_system_design.md`](./save_system_design.md)。

## 游戏概述

- **名称**: WWL Adventure
- **类型**: 吸血鬼幸存者 like 实时生存 Roguelike
- **视角**: 2D 俯视角
- **目标平台**: Web / Android / iOS
- **单局时长目标**: 5-15 分钟
- **核心乐趣**: 走位生存 + 自动战斗 + 升级构筑

## 核心循环

```text
移动走位 → 自动攻击敌人 → 击杀掉落经验/金币 → 拾取经验升级 →
三选一强化 → 武器解锁/升级/选流派 → 敌人变强 → 循环 → HP归零结算
```

## 当前已实现系统

### 玩家

| 属性 | 数值 |
|------|------|
| 初始 HP | 100 |
| 移动速度 | 170 |
| 受击无敌 | 受击后短暂无敌，避免连续扣血 |
| 死亡 | 播放死亡动画后进入结算 |

- 控制方式：键盘 WASD / 方向键 + 触屏虚拟摇杆
- 摄像机跟随玩家
- 头顶血条同步 `GameState.hp_changed`
- 升级时播放脚下光效

### 敌人

| 属性 | 当前基础值 |
|------|------------|
| HP | 12 |
| 移动速度 | 60 |
| 接触伤害 | 5 |
| 伤害冷却 | 1 秒 |
| 经验掉落 | 2 |
| 金币掉落 | 1 |

- 行为：持续追踪玩家，碰撞后造成伤害
- 状态：支持 `slow` 和 `stun`
- 生成：从玩家周围可视区外的视口半对角线 + 80~300 像素环形区域随机生成
- 难度曲线：生成间隔随时间缩短；HP / 伤害按 120 秒翻倍，速度按 300 秒成长并封顶到 1.75 倍
- 资源化：`EnemyData` 已定义，生成器已支持读取 `resources/enemies/`，当前尚未配置敌人 `.tres`

### 武器

当前有 16 种武器，开局自带基础利刃，其余通过升级解锁；生命源泉属于角色强化，不占武器槽。

| 武器 | 定位 | 机制摘要 |
|------|------|----------|
| 基础利刃 | 近战输出 | 朝移动/朝向方向扇形斩击 |
| 弓箭精通 | 弹体输出 | 自动瞄准最近敌人发射箭矢 |
| 天雷引 | 范围爆发 | 随机选敌落雷 |
| 护盾球 | 防御持续 | 环绕玩家旋转并碰撞伤害 |
| 荆棘护甲 | 防御反伤 | 玩家受伤时反弹伤害 |
| 散弹枪 | 弹幕爆发 | 多弹丸扇形射击 |
| 火焰瓶 | 持续范围 | 投掷后留下火场 |
| 冰霜环 | 防御控制 | 周期性范围伤害和减速 |
| 圣光棱镜 | 伤害治疗 | 造成范围伤害并治疗 |
| 毒液罐 | 持续范围 | 投掷后留下毒雾 |
| 地雷 | 陷阱爆发 | 随机布雷，敌人靠近爆炸 |
| 激光笔 | 直线穿透 | 对路径上敌人造成伤害 |
| 回旋镖 | 往返弹体 | 飞出后返回，往返可命中 |
| 电磁链 | 连锁输出 | 在多个敌人间弹跳 |
| 锯片陷阱 | 持续陷阱 | 旋转锯片持续伤害 |
| 火箭背包 | 移动尾焰 | 移动时留下火焰轨迹 |

每种武器：

- 继承 `WeaponBase`
- 有独立 `.tscn` 场景
- 有 `resources/weapons/*.tres` 数据资源
- 默认等级上限 8
- 支持 3 条流派路径，每条路径通过 `WeaponPathLevel` 提供数值 bonus 或 `special_tag`

### 升级系统

经验公式：

```text
exp_required = 15 * 1.2^(level - 1)
```

当前升级类型：

| 类型 | 说明 |
|------|------|
| 角色强化 | 移速、最大 HP、治疗、拾取范围；单局最多占用 6 个强化槽，同类再次出现时叠加等级 |
| 武器解锁 | 从未持有武器中随机出现 |
| 武器流派 | 武器 1 级且未选流派时出现；选择后立即升到 Lv.2 并获得该流派首个强化 |
| 武器强化 | 已选流派后按路径给出下一级效果；无路径武器走硬编码强化 |
| 外部升级 | `DataManager.all_upgrades()` 返回的资源会加入候选池 |

当前内置角色强化：

| 名称 | 效果 |
|------|------|
| 疾风步 | 移动速度 +25 |
| 生命强化 | 最大 HP +30，当前 HP +30 |
| 磁力增幅 | 拾取范围 +30 |
| 生命源泉 | 立即恢复 5 HP；之后每 5 秒自动恢复生命，治疗量随强化等级提升 |

单局最多持有 6 把基础武器和 6 种角色强化。升级候选会过滤已满级武器、满槽后的新武器 / 新强化，并保证同一轮不会重复出现同一个武器 ID。升级面板支持刷新全部候选卡片，也支持跳过本次选择直接继续游戏。

### UI

- 主菜单：开始游戏
- HUD：时间、击杀、金币、HP、EXP、等级、武器栏、强化栏、属性按钮
- 升级选择：三张卡片，显示图标、类型和武器分类
- 暂停菜单：展示当前武器槽位、强化槽位并可返回主菜单
- StatsPanel：查看角色基础状态、已持有强化和武器详情
- GameOver：展示时间、击杀、本局武器和强化
- VirtualJoystick：移动端 / Web 自动显示
